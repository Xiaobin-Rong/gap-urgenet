This is the official implementation of GAP-URGENet, a state-of-the-art (SOTA) universal speech enhancement (USE) model.

GAP-URGENet ranked **1st** in the objective evaluation phase of the **ICASSP 2026 URGENT Challenge**.

The Code and checkpoints will be released soon.

Related repo: [UniPASE](https://github.com/Xiaobin-Rong/unipase)
